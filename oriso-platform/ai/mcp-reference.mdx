## Troubleshooting

### Common Issues

**Issue**: MCP servers not loading
- **Solution**: Restart Claude Desktop/Cursor after configuration changes
- **Check**: JSON syntax is valid (use a JSON validator)
- **Verify**: Node.js is installed (`node --version`)

**Issue**: GitHub token not working
- **Solution**: Regenerate token with correct scopes
- **Check**: Token hasn't expired
- **Verify**: Token has access to the repositories you're querying

**Issue**: Slack bot not posting messages
- **Solution**: Ensure bot is added to target channels (`/invite @your-bot`)
- **Check**: Bot token has `chat:write.public` scope
- **Verify**: Channel IDs are correct

**Issue**: Channel not found
- **Solution**: Use Channel ID, not channel name (e.g., `C01234567` not `#general`)
- **Check**: Bot has `channels:join` permission and is in the channel

### Debug Mode

Enable debug logging for MCP servers:

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_token",
        "DEBUG": "true"
      }
    }
  }
}
```

### Check Logs

**Claude Desktop:**
- macOS: `~/Library/Logs/Claude/`
- Windows: `%APPDATA%\Claude\Logs\`

**Cursor:**
- Check Developer Tools (Help → Toggle Developer Tools)
- Look for MCP-related errors in console

## Security Best Practices

### Token Management

- ✅ Use environment variables or secure vaults
- ✅ Rotate tokens regularly (every 90 days)
- ✅ Use fine-grained tokens with minimal scopes
- ❌ Never commit tokens to git
- ❌ Never share tokens in screenshots/docs

### Slack App Security

- ✅ Restrict bot to necessary channels only
- ✅ Review OAuth scopes periodically
- ✅ Use workspace-specific tokens
- ✅ Monitor bot activity logs

### GitHub Token Scopes

- ✅ Principle of least privilege
- ✅ Use fine-grained tokens when possible
- ✅ Enable token expiration
- ✅ Revoke unused tokens

## Quick Reference

### Configuration File Locations

| Tool | Config Location |
|------|----------------|
| Claude Desktop (macOS) | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Claude Desktop (Windows) | `%APPDATA%\Claude\claude_desktop_config.json` |
| Cursor (macOS) | `~/.cursor/mcp.json` |
| Cursor (Windows) | `%APPDATA%\Cursor\mcp.json` |
| VSCode | `.vscode/settings.json` (workspace) |

### Required Tokens

| Service | Token Format | Where to Get |
|---------|-------------|--------------|
| GitHub | `ghp_...` | GitHub Settings → Developer settings → Personal access tokens |
| Slack Bot | `xoxb-...` | Slack API Apps → Your App → OAuth & Permissions |
| Slack Team ID | `T...` | Slack web URL or workspace settings |
