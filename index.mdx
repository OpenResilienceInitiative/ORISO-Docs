## What is ORISO Platform?

ORISO Platform is a comprehensive online consultation platform built with:

- **Microservices Architecture** - Spring Boot backend services
- **Kubernetes Deployment** - k3s lightweight Kubernetes
- **Authentication** - Keycloak identity management
- **Communication** - Matrix protocol for chat, LiveKit for group video calls
- **Databases** - MariaDB and MongoDB
- **Frontend** - React-based web applications
- **Monitoring** - Status page and health dashboard

**Major Configuration Refactoring - Production Ready:**

- ‚úÖ **ConfigMaps & Secrets**: All services now use Kubernetes ConfigMaps/Secrets - no hardcoded values
- ‚úÖ **DNS-Based Discovery**: Services use Kubernetes DNS names instead of IPs
- ‚úÖ **Configuration Validators**: Services validate required configs on startup with clear error messages
- ‚úÖ **Helm Charts**: MariaDB deployment now uses Helm charts with automatic schema initialization
- ‚úÖ **Update Without Rebuild**: Configuration can be updated via ConfigMaps/Secrets without rebuilding images
- ‚úÖ **LiveKit Integration**: Group video calls support via LiveKit server
- ‚úÖ **Status Page**: Modern React-based status page with professional UI

## Prerequisites

Before you begin, ensure you have:

- ‚úÖ Fresh Ubuntu Server 22.04 LTS (or 20.04)
- ‚úÖ Root or sudo access
- ‚úÖ Public IP address
- ‚úÖ Minimum 4 CPU cores and 8GB RAM (16GB+ recommended)
- ‚úÖ 50GB+ disk space (100GB+ recommended)
- ‚úÖ Domain name (required for HTTPS/SSL setup)
- ‚úÖ DNS access (ability to add A records for subdomains)
